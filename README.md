# Arch Linux Install (Dell XPS 9550)

## Prerequisites 
* Disable secure boot
* Change to AHCI
* Turn off legacy ROM
* Get bootable USB
** Rufus: https://rufus.akeo.ie/
** Use DD mode
** File: http://mirror.umd.edu/archlinux/iso/2017.11.01/

## Initial Instalation
* **f12** to select bootable USB
* Partition Drive
